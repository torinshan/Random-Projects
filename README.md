# Random-Projects

## NCAA D1 Salary Value Calculator

This repository contains an interactive dashboard for calculating and comparing NCAA Division 1 coaching salaries across different states, accounting for taxes and cost of living.

### Features

- Interactive map showing NCAA D1 schools
- Salary comparison calculator with tax considerations
- State-by-state tax calculations
- Cost of living adjustments
- Visual rankings and comparisons

### Available Tools

This repository contains three interactive HTML dashboards:

1. **2_School_Taxes_COL_Comparssion_Tool.html** - School comparison with taxes and cost of living
2. **NCAA_D1_Total_Compensation_V4B_Single_School.html** - Single school total compensation analysis
3. **2_School_Benefits_Total_Comp_Compare_Tool.html** - Comprehensive benefits and total compensation comparison tool

### Local Development

To view locally, simply open any of the HTML files in your web browser.
